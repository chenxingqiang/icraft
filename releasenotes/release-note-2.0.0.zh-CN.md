# 发布说明 - 2.0.0 (2025-12-31)

![iCraft Editor](https://raw.githubusercontent.com/gantFDT/icraft/main/public/images/banner.jpg)

## 编辑器
### 新增功能
- 新增模型自身动画功能，支持模型在布局编辑模式下持久播放以及在动画编辑模式下编排播放
- 新增Mermaid时序图支持，用户可以通过Mermaid语法在布局编辑模式下添加时序图
- 自动创建 Mermaid 时序图动画方案，系统会根据用户添加的时序图自动创建对应的动画方案

### 功能优化
- 棱柱类型元素支持棱数配置

### Bug修复
- 修复辉光后处理效果开启后在透视相机下视角不能旋转的问题
- 修复属性面板滑块输入框偶发修改值跳变的问题
- 修复偶发加载不出编辑器配置的问题
